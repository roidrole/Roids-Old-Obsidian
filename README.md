OldObsidian 1.1
=========
[![ CurseForge](http://cf.way2muchnoise.eu/284764.svg)](https://minecraft.curseforge.com/projects/old-obsidian)

This mod is currently updated for `Minecraft 1.12.1` 

Please verify and report any issues!
